# add-feature

Add a new feature to the Japanese Study Tracker app.

## Instructions

When adding features to this single-file app:

1. Read `index.html` first to understand current structure
2. Add CSS styles in the `<style>` section (lines 7-251)
3. Add HTML elements in the `<body>` section (lines 253-318)
4. Add JavaScript functions in the `<script>` section (lines 320-551)
5. If the feature needs persistence, use localStorage with the existing `STORAGE_KEY` or add a new key
6. Call `render()` after any data changes to update the UI
7. Maintain the GitHub dark theme aesthetic (colors: #0d1117, #161b22, #30363d, #c9d1d9, #39d353)
