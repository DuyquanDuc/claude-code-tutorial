# Japanese Study Tracker

A minimal web app to track Japanese study consistency using a GitHub-style contribution heatmap.

## Usage

Open `index.html` in your browser. No server or dependencies required.

## Features

- **Commit study sessions** - Log what you studied (kanji, grammar, vocab, etc.)
- **Contribution heatmap** - Visual calendar showing your study consistency
- **Streak tracking** - See your current consecutive day streak
- **Edit/Delete commits** - Manage your study history
- **Local storage** - Data persists in your browser

## Data

All data is stored locally in your browser's localStorage under `japanese-study-commits`.
